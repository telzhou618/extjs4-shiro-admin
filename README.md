
### extjs4-shiro-admin 轻量级权限管理框架

### 技术选型
-------------
Extjs4、Spring MVC、Mybatis、Mybatis-Plus、Shiro、Mysql、Maven

### 快速开始
-------------
1. git clone https://github.com/foreyer/extjs4-shiro-admin.git
2. cd ~/extjs4-shiro-admin
3. mvn clean install
4. mvn jetty:run
5. http://localhost:8080
